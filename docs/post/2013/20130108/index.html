Mac CPAN

Posted on Jan 8, 2013

開発用にMac miniを買って、eclipseなどインストールしたところ、perlのスクリプトにJcodeが見えないというエラーが。せっかくだからJcodeをインストール。

bash-3.2$ sudo perl -MCPAN -e shell
cpan[1]> install Jcode
